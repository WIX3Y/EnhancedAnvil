# Enhanced Anvil
Lock enchanting table tiers behind permissions and unlock new possibilities with the Enhanced Anvil. The Enhanced Anvil (with AdvancedEnchantments support) allows applying enchanted books to items, even when the normal anvil says "too expensive".

## Commands
### eanvil
Description: Opens the custom anvil gui \
Example Usage: /eanvil \
Aliases: enhancedanvil \
Permission: enhancedanvil.anvil.cost.bypass \
Note: "/eanvil \<player\>" can be run from console to open the gui for other players


### eareload
Description: Reloads the config file \
Example Usage: /eareload \
Aliases: enhancedanvilreload \
Permission: enhancedanvil.reload
